# 技術スタック

このドキュメントは本プロジェクトで使用している主要な技術（言語、パッケージ、ライブラリ、フレームワーク、ランタイム、開発ツール等）を分類ごとにまとめたものです。バージョンはリポジトリ内の `package.json` と `src-tauri/Cargo.toml` に基づきます。

## 言語

- **TypeScript**: ^5.9.3（フロントエンド主要言語、型安全性の提供）
- **JavaScript**: 実行時スクリプト/ビルドツール上で使用
- **Rust**: 1.77.2（Tauri ネイティブサイドの実装）
- **CSS / Tailwind CSS**: Tailwind によるユーティリティファーストなスタイリング

## ランタイム / 実行環境

- **Tauri (デスクトップアプリ向け)**: tauri 2.10.x（ネイティブバイナリとしてデスクトップで実行）
- **Node.js (ビルド / 開発ツール実行)**: プロジェクトは Node 環境でビルド／開発を行う（package.json の npm スクリプトを利用）
- **pnpm**: `pnpm@10.26.2`（パッケージマネージャ）

## フロントエンドフレームワーク / UI ライブラリ

- **React**: ^19.2.0（UI レンダリング）
- **React DOM**: ^19.2.0
- **@monaco-editor/react**: 組み込みコードエディタ（Monaco）
- **recharts**: ^3.5.1（グラフ表示）
- **xterm + xterm-addon-fit**: ターミナルエミュレーション

## バンドラ / ビルドツール / 開発ツール

- **Vite**: ^7.2.4（開発サーバ・ビルド）
- **@vitejs/plugin-react**: Vite と React の統合
- **TypeScript ビルド**: `tsc -b` を利用した型チェック／ビルド
- **PostCSS / Autoprefixer**: CSS 後処理
- **Tailwind CSS**: ^3.4.19（ユーティリティ CSS）

## Tauri 関連（JS 側）

- **@tauri-apps/api**: Tauri の JavaScript API
- **@tauri-apps/cli**: 開発・ビルド用 CLI（dev / build スクリプトで利用）
- 利用している Tauri プラグイン（JS 側）:
  - `@tauri-apps/plugin-dialog`
  - `@tauri-apps/plugin-fs`
  - `@tauri-apps/plugin-http`
  - `@tauri-apps/plugin-opener`
  - `@tauri-apps/plugin-os`
  - `@tauri-apps/plugin-process`
  - `@tauri-apps/plugin-shell`
  - `@tauri-apps/plugin-store`
  - `@tauri-apps/plugin-updater`

## ネイティブ（Rust）側の主要クレート

- **tauri**: 2.10.x（アプリケーションランタイム）
- **tauri-build**: ビルド時ヘルパー
- **tauri-plugin-\*（複数）**: dialog, shell, fs, http, process, store, opener, os, single-instance, updater
- **serde / serde_json**: シリアライズ/デシリアライズ
- **tokio**: 非同期実行ランタイム（feature: full）
- **reqwest**: HTTP クライアント（stream 機能あり）
- **sysinfo**: システム情報取得
- **zip / tar / flate2**: アーカイブ操作
- **futures-util**: 非同期ユーティリティ

## 開発品質・CI・Lint / フォーマット

- **@biomejs/biome**: フォーマッタ / リンター（`format`, `lint`, `biome:check` スクリプト）
- **husky**: Git フック管理（prepare スクリプト）
- **lint-staged**: ステージされたファイルに対する linters

## パッケージ管理 / スクリプト

- `packageManager`: pnpm@10.26.2
- package.json に定義された主なスクリプト: `dev`（vite）、`build`（tsc + vite build）、`tauri:dev`、`tauri:build` など

## その他ユーティリティ / ライブラリ

- **xterm**: 組み込みターミナル表示
- **recharts**: チャート描画

## 開発ワークフローに関するファイル / 設定

- `tsconfig.*.json`: TypeScript コンパイル設定が存在（型チェック / ビルド）
- `vite.config.ts`: Vite の設定（プラグイン、ビルドターゲット等）
- `tailwind.config.js`: Tailwind の設定
- `tauri.conf.json`: Tauri 設定

## 備考 / 注意点

- フロントエンド（`src/`, `renderer/`）は TypeScript + React + Vite を用いて実装され、ネイティブ機能は Tauri の Rust サイド（`src-tauri/`）で補完されています。
- 依存バージョンは `package.json` および `src-tauri/Cargo.toml` に記載のものを参照しています。追加の開発依存やツールは `devDependencies` に含まれています。
